# ShenLing-Extension

蜃灵助手 SillyTavern 第三方插件。

当前版本：`0.4.0`

## 当前状态

这是第一版最小插件空壳，用于验证：

- GitHub 仓库发布
- SillyTavern 通过 Git URL 安装第三方插件
- 插件 JS/CSS 正常加载
- 扩展设置面板正常显示
- 基础设置可以保存
- 全局设置与当前聊天专属状态分开存储
- 设置页可以查看存储诊断并写入测试值
- 主面板已预留插件级通讯日志入口与日志面板
- 设置页已预留副 API 配置，并支持 OpenAI-compatible 测试连接

## 安装

在 SillyTavern 中打开：

```text
Extensions -> Install Extension
```

填入仓库地址：

```text
https://github.com/dreamdana88/ShenLing-Extension
```
